# 信徒生活須知｜<a href="https://hanafeng2021.github.io/tjc-life-quiz/?v=51">線上練習</a>

GitHub Pages 可直接使用的純前端題庫網站。

## 使用方式
1. 將本資料夾全部檔案上傳到 GitHub repository 根目錄。
2. Commit changes。
3. 到 Settings > Pages，Branch 選 main / root。
4. 等 1-3 分鐘後打開 GitHub Pages 網址。

## 檔案
- index.html：主頁
- app.js：練習、模擬考、錯題本、統計功能
- style.css：手機版樣式
- questions.json：題庫資料

題庫依《信徒生活須知》整理，實際教導請以教會正式文件為準。
